# Forked Fates Demonstration Script

**Target length:** 2 minutes 40 seconds  
**Default path:** Clearly labeled Recorded mode  
**Approved pair:** `recorded-original-v1` and `recorded-alternate-v1`  
**Demo promise:** Change one belief. Watch a different story emerge.

This is the locked Phase 0 script. Later phases may tighten timings and replace a Recorded segment with a verified Live run, but they must preserve the proof beats below. Recorded playback must never be described as live generation.

## Operator safeguards

- Start from a fresh session and verify the Recorded badge before narration.
- Do not reveal Director View world truth during the briefing.
- Do not imply that the approved Recorded actions were generated live.
- If Live mode is demonstrated in a later checkpoint, show one genuine decision context and retain this Recorded pair as the recovery path.
- Pause only at completed-turn boundaries.
- Before recording, verify the Original still ends at turn 12, the Alternate at turn 7, and the fork hash matches through turn 2.
